A full-stack app to track a user's accomplishments and statistics in Apex Legends and Overwatch.

![stat_tracker](stat_tracker.PNG)

## Technology
	
* HTML
* JavaScript
* CSS
* Third Party APIs
* Axios
* Custom Server-Side API
* Vue
* Express
* Node.js
* Concurrently
* Morgan
* Nodemon


## Install

```sh
npm install
```

## Usage

```sh
npm run dev
```
